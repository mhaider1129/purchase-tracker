# Purchase Frontend (Vite)

This project is built with [Vite](https://vitejs.dev/) and React.

## Available Scripts

In the project directory, you can run:

### `npm start`

Starts the development server (alias of `npm run dev`).
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### `npm run dev`

Runs the Vite dev server with hot module replacement.

### `npm run test`

Runs the Vitest test suite in a Node/jsdom environment.

### `npm run build`

Builds the app for production into the `dist` folder.

### `npm run preview`

Serves the production build locally for inspection.

### `npm run lint`

Lints the source files with ESLint.

### `npm run format`

Formats source and config files with Prettier.
